# Figma Essentials for Graphic Design (20% for 80% Mastery)

## 1. **Interface Basics**
- **Canvas**: Workspace for designing (infinite zoom/pan).
- **Layers Panel**: Manage design elements hierarchy.
- **Toolbar**: Access core tools (Move, Shapes, Text, Pen).
- **Properties Panel**: Adjust alignment, spacing, effects, and constraints.

## 2. **Core Tools**
- **Shapes**: Rectangles, circles, polygons, lines.
- **Text Tool**: Add/edit text with font/style controls.
- **Pen Tool**: Create custom vector paths.
- **Boolean Operations**: Combine shapes (Union, Subtract, Intersect).
  ```figma
  Select shapes → Right-click → "Boolean Operation"
  ```

## 3. **Components & Variants**
- **Components**: Reusable design elements (buttons, icons).
  - Create: Select object → `⌘/Ctrl + Alt + K`
  - Override: Swap text/colors in component instances.
- **Variants**: Manage component states (hover, active, disabled).
  ```figma
  Select component → "Variants" in Properties Panel.
  ```

## 4. **Auto Layout**
- **Dynamic Layouts**: Automatically adjust spacing/sizing.
  ```figma
  Select elements → "Auto Layout" in Properties Panel.
  ```
- **Padding/Spacing**: Set margins between elements.

## 5. **Constraints**
- **Responsive Design**: Define how elements resize.
  - Options: Left/Right, Top/Bottom, Scale, Fixed.

## 6. **Prototyping Basics**
- **Link Frames**: Create interactive flows.
  ```figma
  Select object → "Prototype" tab → Drag to target frame.
  ```
- **Transitions**: Set animation type (Smart Animate, Slide, Dissolve).

## 7. **Collaboration**
- **Comments**: `C` to add notes for feedback.
- **Share**: Invite via link → Set permissions (View/Edit).
- **Team Libraries**: Publish components for shared access.

## 8. **Shortcuts**
- `F`: Frame tool
- `R`: Rectangle
- `T`: Text tool
- `⌘/Ctrl + D`: Duplicate
- `⌘/Ctrl + G`: Group
- `⌘/Ctrl + /`: Search tools

## 9. **Color & Styles**
- **Color Picker**: Use hex codes or eyedropper.
- **Styles**: Save colors/text/effects for reuse.
  ```figma
  Select element → "Style" icon → Create/Apply.
  ```

## 10. **Exporting Assets**
- Select layers → "Export" tab → Choose format (PNG, SVG, PDF).
- Use `@1x`, `@2x`, `@3x` for resolution scaling.

## 11. **Best Practices**
- Name layers clearly (Avoid "Rectangle 43").
- Use grids (Layout Grids) for alignment.
- Stick to 4px/8px spacing multiples.
- Organize with frames (not just groups).

## 12. **Plugins & Resources**
- **Plugins**: Iconify (icons), Unsplash (images), Figmotion (animation).
- **Templates**: Community → Search "UI Kits", "Wireframes".
